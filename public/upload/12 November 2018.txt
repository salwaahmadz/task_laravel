Query adalah sebuah proses untuk meminta data dari database.

Sintak	

	SELECT	list
	from	tables
	where	condition

perintah select secara khusus pada kolom kolom yang akan diambil (sesuatu yang luas)

Contoh 

	Select employee, date, salary
	From	Emp
	Where	Salary=2row;


Nested Query

Sub Query adalah sebuah pernyataan select didalam perintah select. Pernyataan select
berdiri dari 1 atau lebih sub query yang dikenal dengan nested query atau nested select

Manfaat Sub Query

Sub Query sangat berguna ketika user memerlukan untuk menampilkan rows/ select rows dari
sebuah table dengan kondisi yang tergantung pada data dalam table

Single Row Sub Query
Contoh

Select	MIN(salary)
FROM	Emp;
Select	MAX(salary)
FROM	Emp;
